Utilizacion del código:
    1. Definir un problema con la clase Problema Clasificacion de manera manual.
         Existe una plantilla para facilitar la definición.
    2. Asignar la lista de predicados del problema del archivo auxiliares
    3. Pasar por parametro a la función app del fichero app.py el problema y la lista de objetivos
    4. Introducir parametros por consola, segun lo que se desee ejecutar

Los ejemplos usados en los experimentos se pueden ejecutar en los archivos:
    1. rueda.py
    2. bloques.py
    3. problema_habitaciones.py
    4. buceadores.py
